# SSA Front-end Boilerplate

React boilerplate with the requested stack, ready for wizard implementation later.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run test` - run tests
- `npm run lint` - run ESLint

## Included stack

- React.js (Vite)
- Ant Design + Less
- Context API
- React Router
- react-i18next
- Optional testing (React Testing Library + Vitest)
